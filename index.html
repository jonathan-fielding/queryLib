{ queryLib }

queryLib is a simple library that loops through the stylesheets you have included in the page finding any media queries and parsing any that contain min-width and max-width values, returning a object containing a list of these media queries.

Download queryLib alpha

Get started

To get started with queryLib the first step is to include it in your project, you can do this in two ways

  • Download and add the queryLib.js (or queryLib.min.js) file to your site
  • Use Bower to add to your project using bower install querylib

API

At the moment the API is pretty basic

Get mediaqueries (async)

To get the mediaqueries we need to do an asyncronous request go get all the attached stylesheets and then parse them in javascript, our results are returned to the callback.

queryLib(function(queries){
    console.log(queries)
});

Get mediaqueries (cached)

To retrieve the last set of media query results you can look at the queryLib.cached object, please note that we must have done a async request earlier to populate the cache.

queryLib.cached
